# **Family Account**

## **Overview**

Simple and light-weight account management app for iOS.

## **Features**

- Any user data, logs and activities are not collected.
- User data are saved in protected directory.
- Free download and NO ads.

## **Requirements**

- macOS Big Sur 11.2.3+
- Xcode 12.5+
- Swift 5.4+
